# Description
- **프로젝트 소개** <br>
  해당 프로젝트는 TPS 플랫폼 게임입니다. <br>
  여러 아이템과 NPC 시스템, 다양한 플랫폼 등 플랫폼 게임의 다양한 기능들을 담았습니다. <br>
  작업간에 코드 재사용성을 높이고 유지보수가 용이한 쪽으로 작업해보는 것을 목표로 개발했습니다.<br>
<br>

- **개발 기간** : 2025.03.05 - 2025.03.11
- **사용 기술** <br>
-언어 : C#<br>
-엔진 : Unity Engine <br>
-개발 환경 : Windows 11 <br>
<br>
